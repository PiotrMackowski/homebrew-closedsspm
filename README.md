```bash
brew update

```

## Uninstall

```bash
brew uninstall closedsspm
```

This tap is automatically maintained by [GoReleaser](https://goreleaser.com/). Formulas are updated on each [ClosedSSPM release](https://github.com/PiotrMackowski/ClosedSSPM/releases).
